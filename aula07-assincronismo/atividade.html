Atividade

  1. Elabore uma pesquisa comparando XmlHttpRequest, fetch, Promises e async/await
  2. Crie um novo programa que implemente o ajax3.html usando fetch
  3. Crie um novo programa que implemente o ajax3.html usando promises
  4. Crie um novo programa que implemente o ajax3.html usando async/await

Para cada um dos programas, faça uma comparação de desempenho e facilidade de uso entre os métodos utilizados.

A pesquisa e os códigos devem ser salvos em um repositório no Github, cujo endereço deverá ser entregue na atividade de entrega no Moodle.

O repositório deve conter os códigos e as informações da pesquisa devem estar no README.md. Use markdown para formatar a resposta.

O repositório deve ser público.

Esta atividade pode ser realizada em grupo de 3 a 5 participantes.
